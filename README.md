# PrepWise | Advanced AI Interview Preparation

PrepWise is a high-stakes mock interview platform powered by AI to help candidates master their professional presence.

## Tech Stack & Languages

- **TypeScript**: The core language used for both frontend and server-side logic, ensuring type safety and reliability.
- **Next.js 15 (React 19)**: The framework used for building the performant, SEO-friendly App Router architecture.
- **Tailwind CSS**: Used for all styling, including custom "sudden" animations, "letter-reveal" entrance effects, and responsive glassmorphism UI.
- **Firebase**: 
  - **Firestore**: A NoSQL database for storing user profiles, interview sessions, and feedback.
  - **Firebase Authentication**: Secure login via Email/Password and Google.
- **Genkit**: The AI orchestration layer used to build Aria's adaptive reasoning, dynamic question generation, and comprehensive feedback audits.
- **ShadCN UI**: A collection of high-quality, accessible components used for the platform's professional interface.
- **Lucide React**: The library for all system iconography.

## Key Features
- **Aria Persona**: A professional AI interviewer with human-like acknowledgment logic.
- **Biometric Analytics**: Simulated focus and confidence tracking using computer vision logic.
- **Dynamic Letter-Reveal**: A sophisticated entrance animation for brand headings.
- **Admin Console**: Secure dashboard for user and payment oversight.
