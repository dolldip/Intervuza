# **App Name**: AssessAI

## Core Features:

- User Authentication and Management: Secure email/password and OTP login with role-based access (User/Admin) and a 'Delete my data' option to manage user accounts.
- Profile and Resume Parsing: Users create profiles with job role, experience, and skills. Uploaded resumes are parsed via AI to extract relevant information.
- AI Interview Engine: Conduct mock interviews with dynamically generated questions based on user profile, resume, and job description; difficulty adapts to user performance. Stress interview mode available.
- Multimodal Answer Analysis: AI analyzes answers using text, voice, and camera inputs to assess relevance, tone, confidence, eye contact, and emotional expression, and tool use.
- Feedback Generation: Provide question-specific feedback including strengths, weaknesses, and sample answers. Comprehensive feedback includes overall, confidence, and role-fit scores; body language & emotion report.
- AI Interview Coach: Generate personalized improvement plans with daily practice suggestions, targeted drills, and progress tracking.
- Interview Recording and Playback: Save and replay interviews with timeline-based feedback comments and a performance history dashboard.

## Style Guidelines:

- Primary color: HSL(210, 65%, 50%) - A professional blue (#3399FF) that inspires confidence and reliability.
- Background color: HSL(210, 20%, 95%) - A very light, desaturated blue (#F0F8FF) that provides a clean and unobtrusive backdrop.
- Accent color: HSL(180, 55%, 40%) - A contrasting teal (#33CCCC) to highlight key interactive elements and CTAs, ensuring they stand out.
- Headline font: 'Space Grotesk', a proportional sans-serif with a techy feel; use 'Inter' sans-serif for body text
- Use professional and clear icons to represent different features and functionalities.
- Maintain a clean and organized layout with a focus on ease of use and quick access to information.
- Incorporate subtle transitions and animations to improve user engagement without distracting from the content.
